# E-Commerce-Sales-Performance-Dashboard-Power-BI
Interactive E-Commerce Sales Performance Dashboard built using Microsoft Power BI, featuring KPI analysis, sales trends, category performance, and regional insights using DAX and Power Query. 
This project showcases an interactive Sales Performance Dashboard built entirely using Microsoft Power BI. The dashboard analyzes key business metrics and provides insights into sales performance, product trends, and regional distribution.

**Project Workflow**
Data Cleaning using Power Query

Data Modeling within Power BI

KPI Creation using DAX

Interactive Dashboard Design

Total Quantity Sold

**Objective**

The goal of this dashboard is to provide clear insights into sales performance and customer purchasing behavior, enabling better business decision-making
 Sales Trend Analysis: Weekly sales visualization revealed fluctuations in revenue patterns, helping identify high-performing periods and potential seasonal demand trends.

* Category Performance: Analysis of sales by category highlighted the top-performing product segments contributing the highest share of total revenue.

* Regional Sales Distribution: Geographic analysis of sales by state revealed key regions generating the highest revenue, supporting better market focus and regional strategy.

* Customer Purchasing Behavior: The Average Order Value (AOV) metric helped evaluate customer spending patterns and overall purchasing efficiency per order.

* Payment Method Insights: Payment distribution analysis showed customer preference for specific payment methods, providing useful insights for improving checkout experience and payment strategy
